      About Studio & Development Objectives

      Detailed project specs and developer documentation.

      Project Objective

      The Speech Emotion Recognition (SER) Studio serves as an interactive full-stack AI project to demonstrate real-time audio feature engineering, digital signal processing (DSP), deep learning inference, and dynamic explainability. By coupling speech physics (energy levels and pitch frequencies) with spectrogram pattern recognition, it achieves robust, low-latency mood classification.

      Technologies Used

      • Frontend Web App: HTML5, Vanilla CSS3 Grid, Web Audio API
      • Digital Signal Processing: Cooley-Tukey Radix-2 FFT, Hann Windowing
      • Backend API: Python 3 Flask, Scikit-Learn, NumPy, SciPy
      • Deep Learning Library: TensorFlow 2.x (Keras)

      Future Improvements

      • Multimodal Fusion: Integrating facial tracking (Camera CNN) with voice analysis.
      • On-Device Inference: Migrating the backend ensemble directly to TensorFlow.js in-browser.
      • Real-time Noise Subtraction: Adaptive spectral gating nodes in Web Audio API.
